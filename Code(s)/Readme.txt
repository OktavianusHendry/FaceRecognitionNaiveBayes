Cara run:
1. Tambahkan foto sebanyak minimal 20 pada folder image_data - Your Face
2. Rename folder Your Face jadi nama anda
3. Install library Python OpenCV 
4. Run file face_trainer.py untuk training dataset
5. Coba Run file face_recognise.py untuk melihat apakah muka sudah terdeteksi
6. Untuk melakukan komunikasi Python + Arduino, gunakan Serial Communication dengan library python yaitu pyserial
7. Upload kode pada folder Unlock_RFID_Final ke ESP32 dan buka Serial Monitor untuk memastikan ESP32 sudah terkoneksi dengan WiFi
8. Tutup Serial Monitor
9. Buka Folder Kode pada IDE dan Run file face_recogniser1.py dan ganti COM Port pada line ke 6 sesuai dengan Port yang digunakan
10. Selesai
